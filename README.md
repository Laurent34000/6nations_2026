
# 🏉 Six Nations 2026 – Data Analysis Project

This project is a data analysis of the **Six Nations 2026** tournament, combining web scraping, data processing, and visualization.

As a rugby enthusiast, I built this project to analyze player and team performance using real data scraped from ESPN and Rugbyrama.

---

## 🚀 Objectives

* Collect data from online sources
* Clean and structure datasets
* Analyze team standings and player performance

---

## 🌐 Data Sources

* [ESPN](https://www.espn.com/rugby/standings/_/league/180659/sort/gameswon/dir/desc)
* [Rugbyrama](https://www.rugbyrama.fr/resultats/rugby/6-nations/statistiques/joueurs)

---

## 🛠️ Tech Stack

* BeautifulSoup
* Python
* Pandas
* Matplotlib
* Jupyter Notebooks

---

## ▶️ How to Run
1. Clone the repo
2. Install requirements
3. Open notebooks
